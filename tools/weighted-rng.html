How it works

Pick Random (O(n) time, O(1) space)

Only one random number is required to determine the index of the selected label. The algorithm uses a cumulative sum approach:

  1. Get total of all weights.
  2. Generate rng = random(0, total_weight).
  3. For each label:
    sum += weight
    if rng < sum: return current_label

Plot Distribution (O(s*log(n)) time, O(n) space)

For a sample size of 10,000, iterating O(n) every time is slow. Instead, we pre-calculate an array of cumulative sums and use binary search:

  1. Pre-calculate array acc containing cumulative weights.
  2. Loop 10,000 times:
    Generate rng = random(0, total_weight).
    index = binarySearch(acc, rng)
    results[index].frequency++
